## Why this repo stands out
- Interview-ready roadmap across backend, AI, and system design.
- Real-world topics: RAG, LangChain agents, MCP, pgvector, LLM guardrails.
- Structured progression: basics -> intermediate -> advanced -> QnA.
- Practical prep files for self-intros, project storytelling, and company-specific interview strategy.

### Agentic AI stack
- OpenAI API fundamentals and prompting patterns
- LangChain chains, tools, and agents (LangGraph multi-agent workflows)
- RAG design and retrieval workflows (chunking, pgvector, hybrid search, reranking)
- pgvector and vector similarity search
- LLM guardrails, evaluation strategy (RAGAS, LangSmith, Langfuse), and safety
- MCP server and tool-integration concepts

### Full-stack + infrastructure
- Python asyncio, type hints, dataclasses, metaclasses
- Redux Toolkit, Zustand, React Query — client & server state management
- Redis — caching, rate limiting, pub/sub, distributed locks, sessions
- Advanced MongoDB — aggregation pipelines, transactions, sharding, change streams
- Advanced Express — async patterns, WebSockets, security hardening
- Kubernetes basics, Docker multi-stage builds, GitHub Actions CI/CD
- AI tools ecosystem — Claude Code, Cursor, Windsurf, MCP, CLI tools

| Module | What you learn |
|---|---|
| 01_python | Python fundamentals, OOP, decorators, generators, asyncio, type hints, dataclasses, metaclasses |
| 02_flask | Flask APIs, JWT auth, blueprints, app factory, SQLAlchemy, SocketIO |
| 03_openai_langchain | OpenAI API, LangChain chains/memory/tools, advanced agents, LangGraph multi-agent |
| 04_rag_pgvector | End-to-end RAG pipeline, pgvector SQL, hybrid search, reranking, evaluation |
| 05_javascript_node | JavaScript, Node.js event loop/streams, Express advanced, TypeScript |
| 06_mongodb_mongoose | MongoDB aggregation, indexing, transactions, schema patterns, sharding |
| 07_react | React hooks, Router v6, Suspense, testing with RTL/Jest, React Query |
| 08_sql | SQL JOINs, CTEs, window functions, JSONB, full-text search, partitioning, EXPLAIN |
| 09_rest_security | REST design, API security, OAuth2, PKCE, XSS/CSRF/SSRF prevention, OWASP Top 10 |
| 10_dsa | 32 problems (Python + JS), Big-O, all patterns (sliding window, DP, graphs), Segment Tree, LFU Cache |

| 19_state_management | Redux Toolkit (createSlice/Thunk/EntityAdapter), Zustand middleware, React Query, XState |

| 21_ai_tools_trending | Claude Code, Cursor, Windsurf, Lovable, MCP protocol, CLI tools, Git hooks, LangGraph |
